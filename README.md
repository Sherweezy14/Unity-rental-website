# Unity Event Hall Website

## Overview

The Unity Event Hall Website is a modern, responsive marketing website built for Unity Event & Banquet Hall in Boston, Massachusetts. The website is designed to showcase the venue, highlight available event spaces, display rental pricing, and generate customer inquiries for weddings, repasts, corporate events, baby showers, cultural events, and private celebrations.

The primary goal of the project was to create a professional online presence that increases venue visibility, improves customer trust, and generates rental leads.
